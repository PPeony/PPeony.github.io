leetcode1111. 有效括号的嵌套深度

示例 1：
输入：seq = “(()())”
输出：[0,1,1,1,1,0]

示例 2：
输入：seq = “()(())()”
输出：[0,0,0,1,1,0,1,1]
解释：本示例答案不唯一。
按此输出 A = “()()”, B = “()()”, max(depth(A), depth(B)) = 1 。
像 [1,1,1,0,0,1,1,1]，也是正确结果，其中 A = “()()()”, B = “()”, max(depth(A), depth(B)) = 1 。

官方题解有一处不是很懂，说要按照奇偶层划分，只能换一个思路。
比较A和B的深度，对于左括号，如果A的深度大了，那么必须增加B的深度；相反B的深度大了必须增加A的深度。右括号是用来降深度的，如果A深度大，这个右括号就给A，反之就给B。最后的目的就是为了让AB在同一水平线上，谁也别超过谁太多。
