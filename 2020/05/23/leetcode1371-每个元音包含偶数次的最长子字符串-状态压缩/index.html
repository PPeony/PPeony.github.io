leetcode1371. 每个元音包含偶数次的最长子字符串

给你一个字符串 s ，请你返回满足以下条件的最长子字符串的长度：每个元音字母，即 ‘a’，’e’，’i’，’o’，’u’ ，在子字符串中都恰好出现了偶数次。

示例 1：
输入：s = “eleetminicoworoep”
输出：13
解释：最长子字符串是 “leetminicowor” ，它包含 e，i，o 各 2 个，以及 0 个 a，u 。

示例 2：
输入：s = “leetcodeisgreat”
输出：5
解释：最长子字符串是 “leetc” ，其中包含 2 个 e 。

示例 3：
输入：s = “bcbcbc”
输出：6
解释：这个示例中，字符串 “bcbcbc” 本身就是最长的，因为所有的元音 a，e，i，o，u 都出现了 0 次。

提示：

  • 1 <= s.length <= 5 x 10^5
  • s 只包含小写英文字母。

状态压缩

本题没思路，做了好几遍状态压缩也没反应，只能看别人题解，写下自己理解。
在字符串当中如果 [l,r] 是最长的符合题意的，那么 [0,r]-[0,l-1] 的状态就等于 [l,r] 的状态。所以在一次遍历的过程当中，把现在的状态记录下来，存储到hash表当中（可以是map，也可以是数组），后来遍历的时候，直接取出对应的状态就行。
对于奇偶情况，能用二进制表示，奇数为1，偶数为0，一共只有32种情况。
